# CLEO

==Installation Manual for UniServer==
1. Download and save the latest server file(for example 14_0_3_ZeroXIV.exe) to drive C:
2. The file is a self-extracting archive, double click to run the extractor.
3. A new folder UniServerZ is created containing UniController.exe and support folders and files.
4. Start controller by double clicking on file UniController.exe
5. Click Start Apache button. Indicator changes from red to green when server is running.
6. Click Start MySQL button. Indicator changes from red to green when server is running.
7. Display test page by clicking button View www or type http://localhost into your browser.
